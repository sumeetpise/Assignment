1.Dependencies Required
Python 3.x
pytest (pip install pytest)
requests (pip install requests)


2.Running the tests
To run the tests, open cmd and navigate to the directory where you have saved the file and run the command:

* py.test 
 
or

* py.test -v -s (here -v stands for more information and -s stands for logs in output)

or

* py.test --html=report.html (generates html report)
